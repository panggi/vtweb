# Vtweb

Ruby Wrapper for Veritrans VT-Web. Visit https://www.veritrans.co.id for more information.

## Installation

Add this line to your application's Gemfile:

    gem 'vtweb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vtweb
